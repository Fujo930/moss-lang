# Moss identity

![Moss branching M mark](../src/mosslang/studio_assets/moss-mark.svg)

Moss is a programming language for software that must remain understandable
while humans and AI agents change it over a long lifetime.

## Public description

Use this short description:

> Moss is an AI-designed and AI-built programming language prototype for
> long-lived AI-and-human software projects.

Moss `0.3.0-alpha` is a verified-frontend self-hosting preview. Its Moss-written
frontend is recursively compared against the Python host frontend, while the
active interpreter and CLI still run on Python.

## Mark

The branching M is the primary Moss mark. Its two lower paths represent human
and AI contributors. They meet in a shared syntax tree and continue upward as
one checked project. The three endpoints also suggest source, analysis, and
execution.

Use `src/mosslang/studio_assets/moss-mark.svg` as the canonical asset. Keep its
shape and colors intact, leave clear space around it, and do not place text
inside the mark.

## Colors

- Moss green `#71D6A2`: language and successful analysis
- Build gold `#F2C14E`: execution, active work, and self-hosting progress
- Trace blue `#8BD9F7`: inspection and tooling
- Ink `#121916`: stable high-contrast foundation

The palette deliberately combines several signals instead of making every
surface green. Moss tools should feel precise, calm, and work-focused.

## Voice

Prefer concrete claims supported by commands and examples. Say "self-hosting
preview" rather than "fully self-hosted", and "AI-designed and AI-built by
Codex in collaboration with Fujo930" rather than implying no human direction.
