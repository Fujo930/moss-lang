Moss 0.5.0 for Windows

Run Moss-0.5.0-Windows-Setup.exe to install the Moss compiler and Studio.
The installation also includes moss-lsp for editor integrations.

Moss-0.5.0-Windows-Portable.zip contains the same compiler, language server,
and Studio without an installer.

Installed components:
- moss.exe command-line compiler
- Moss Studio desktop launcher
- Start menu shortcuts
- optional desktop shortcut
- optional user PATH entry
- Windows uninstall entry

Default install folder:
%LOCALAPPDATA%\Programs\Moss

Default Studio workspace:
%USERPROFILE%\Documents\Moss Workspace

The installed application includes its own runtime and does not require Python.
