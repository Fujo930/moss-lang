# 2026-06-08 — v0.5.6 released

## Summary
Unified all CLI commands onto bytecode VM. 116 tests pass, selfhost 5/5.

## Fixed bugs
- **Short-circuit and/or**: Compiler now emits jump-based short-circuit.
  Root cause of selfhost crash (while index<len and chars[index] failing).
- **textJoin**: Fixed (parts,sep) order and 1-arg support matching Runtime.
- **Import path resolution**: VM now uses import_paths for multi-root projects.
- **Imported tests**: _load_imports now merges tests from imported modules.
